# LoganCreations

A static GitHub Pages / iPhone Home Screen creative studio app.

## Current version

This first version includes:

- Mobile-first main dashboard
- App-like Home Screen metadata
- Module buttons for:
  - Create Sprite
  - Sprite Factory
  - Book / Novel Generator
  - Culture Generator
  - Language / Name Engine
  - 3D Model Generator
  - Monster Sound Generator
  - Comic Generator
- Interactive module preview panel
- Light/dark toggle
- SVG app icon
